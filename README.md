# Maermok Studio

Локальная музыкальная библиотека на React/Vite. Треки сохраняются в браузере через IndexedDB и могут работать без интернета.

## Важно про сохранения

Браузер хранит данные отдельно для каждого адреса. `127.0.0.1:5173`, `localhost:5173` и `192.168.x.x:5173` считаются разными местами.

Для себя открывайте один постоянный адрес:

```text
http://127.0.0.1:5173/
```

Для друзей используйте IP вашего компьютера в локальной сети:

```text
http://ВАШ-IP:5173/
```

У каждого друга будет своя локальная библиотека в его браузере. Чтобы передать свою библиотеку, используйте экспорт/импорт JSON в настройках приложения.

## Запуск для себя

```bash
npm run dev
```

Откройте `http://127.0.0.1:5173/`.

## Запуск для друзей в одной сети

```bash
npm run share
```

Узнайте IPv4-адрес компьютера:

```powershell
ipconfig
```

Дайте друзьям ссылку вида `http://192.168.1.25:5173/`. Компьютер с проектом должен быть включен, а сервер должен продолжать работать.

## Запуск без интернета

Если зависимости уже установлены в `node_modules`, интернет не нужен:

```bash
npm run offline
```

Для друзей без интернета в одной сети:

```bash
npm run offline:share
```

## Сборка

```bash
npm run build
```

## Хостинг

Проект готов для Netlify и Vercel.

Netlify:

- Build command: `npm run build`
- Publish directory: `dist`

Vercel:

- Framework: `Vite`
- Build command: `npm run build`
- Output directory: `dist`

После публикации данные будут храниться отдельно на новом домене. Чтобы перенести свою библиотеку с локальной версии, сделайте экспорт JSON в настройках приложения и импортируйте его на опубликованном сайте.
